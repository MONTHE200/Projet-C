# Projet S5 

 Manipulation de Grands Entiers Binaires 

 ## Fonctionnalités

- Comprendre la représentation d’un grand entier binaire stocké dans un tableau dynamique.

- Implémenter les opérations arithmétiques de base sur ce type :
(addition, soustraction, comparaison, affichage, division par 2)

- Implémenter des algorithmes avancés :
   PGCD (algorithme binaire d’Euclide)
    calcul du modulo sans division
    multiplication égyptienne
     exponentiation rapide modulaire
